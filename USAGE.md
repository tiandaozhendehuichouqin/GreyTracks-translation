# 使用指南

## 🎉 新功能已实现！

### 1. ✅ 划词显示翻译按钮
- 选中英文文本后，会在旁边显示一个紫色渐变的翻译图标
- 点击图标即可进行翻译
- 点击其他地方会自动隐藏图标

### 2. ✅ 右键菜单翻译
- 选中文本后右键
- 点击"翻译选中文本"
- 直接在旁边显示翻译结果

### 3. ✅ 透明翻译卡片
- 翻译结果显示在透明小框框中
- 长宽根据翻译内容自动调整
- 支持拖动
- 自动隐藏（可配置）

## 📖 使用方法

### 方法一：划词 + 点击按钮（推荐）

1. **选中英文文本**
   - 在任意网页上用鼠标选中要翻译的英文

2. **点击翻译图标**
   - 选中后会自动出现一个紫色渐变的圆形图标
   - 图标上有翻译标志
   - 点击图标开始翻译

3. **查看翻译结果**
   - 翻译结果会显示在旁边的白色卡片中
   - 卡片会自动调整大小以适应内容
   - 可以拖动卡片查看
   - 5 秒后自动消失（可在设置中调整）

### 方法二：右键菜单

1. **选中英文文本**
   - 在任意网页上选中要翻译的文本

2. **右键点击**
   - 在选中的文本上点击右键

3. **选择"翻译选中文本"**
   - 在右键菜单中点击此选项
   - 翻译结果会立即显示

### 方法三：键盘快捷键

1. **选中英文文本**

2. **按 Ctrl/Cmd + Shift + T**
   - 立即触发翻译

## ⚙️ 设置选项

### 打开设置

- 点击浏览器工具栏的插件图标
- 点击"设置"按钮

### 可用选项

1. **启用划词翻译**
   - 开启/关闭整个插件功能

2. **目标语言**
   - 选择要翻译成的语言
   - 默认：简体中文

3. **源语言**
   - 选择原文的语言
   - 默认：英语
   - 可选：自动检测

4. **显示翻译按钮**
   - 开启：选中文本后显示按钮，点击才翻译
   - 关闭：选中文本后自动翻译
   - 默认：开启（推荐）

5. **自动隐藏延迟**
   - 翻译卡片自动消失的时间
   - 默认：5000 毫秒（5 秒）
   - 设为 0 则不自动隐藏

6. **翻译服务**
   - LibreTranslate（免费）
   - 自定义 API

7. **清除缓存**
   - 一键清除所有翻译缓存

## 🎨 界面说明

### 翻译按钮
- 紫色渐变圆形图标
- 大小：32x32 像素
- 显示在选中区域右侧
- 悬停时会放大
- 点击后立即翻译

### 翻译卡片
- 白色背景，圆角设计
- 原文区域：灰色背景，左侧紫色边框
- 译文区域：淡色渐变背景，左侧紫色边框
- 加载时显示旋转动画
- 支持滚动（内容过长时）
- 可拖动位置

## 🔧 故障排除

### 问题 1：选中文字后没有显示按钮

**解决方案：**
1. 检查插件是否已启用（点击插件图标查看开关）
2. 检查是否在输入框内（输入框内不触发）
3. 刷新网页后重试
4. 在插件设置中确认"显示翻译按钮"已开启

### 问题 2：点击按钮后无法翻译

**解决方案：**
1. 检查网络连接
2. 查看浏览器控制台错误（F12）
3. 在设置中更换翻译 API 端点
4. 尝试使用其他翻译服务

### 问题 3：翻译结果显示错误

**可能原因：**
- 翻译 API 不可用
- 网络连接问题
- 文本过长

**解决方案：**
1. 缩短要翻译的文本
2. 在设置中配置可用的 API 端点
3. 检查 LibreTranslate 服务状态

### 问题 4：翻译卡片不自动消失

**解决方案：**
1. 在设置中检查"自动隐藏延迟"配置
2. 如果设为 0，则不会自动隐藏
3. 手动点击卡片外部即可关闭

## 💡 使用技巧

1. **快速翻译**
   - 设置"显示翻译按钮"为关闭
   - 选中文本后自动翻译，无需点击

2. **保持翻译结果**
   - 将"自动隐藏延迟"设为 0
   - 翻译结果会一直显示，直到点击其他地方

3. **拖动卡片**
   - 可以拖动翻译卡片到任意位置
   - 方便对照原文和译文

4. **缓存机制**
   - 相同的文本不会重复翻译
   - 提高响应速度，节省流量

## 🌐 翻译 API 配置

### 使用 LibreTranslate 公共实例

默认配置：
```
端点：https://libretranslate.com/translate
```

如果不可用，可以：
1. 查找其他 LibreTranslate 公共实例
2. 或自行部署 LibreTranslate

### 自定义 API

如果有自己的翻译 API：
1. 在设置中选择"自定义 API"
2. 输入完整的 API 端点 URL
3. 确保 API 兼容 LibreTranslate 格式
