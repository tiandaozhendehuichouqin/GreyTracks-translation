# 更新日志

## [1.0.0] - 2026-03-08

### ✨ 新增功能

#### 1. 划词显示翻译按钮
- 选中英文文本后自动显示紫色渐变翻译图标
- 图标显示在选中区域右侧
- 点击图标触发翻译
- 点击其他地方自动隐藏图标
- 悬停和点击动画效果

#### 2. 翻译结果卡片优化
- 透明卡片设计，白色背景
- 根据内容自动调整大小
- 支持拖动功能
- 添加加载动画（旋转 spinner）
- 自动隐藏功能（可配置延迟时间）
- 原文和译文分区显示
- 支持滚动（内容过长时）

#### 3. 右键菜单翻译
- 选中文字后右键显示"翻译选中文本"选项
- 点击后直接在旁边显示翻译结果
- 与划词按钮翻译共享相同的卡片组件

#### 4. 配置选项
- 新增"显示翻译按钮"选项
- 可选择自动翻译或点击按钮翻译
- 默认开启显示按钮（推荐）

### 🎯 使用方式

#### 方法一：划词 + 点击按钮（推荐）
1. 选中英文文本
2. 点击出现的翻译图标
3. 查看翻译结果

#### 方法二：右键菜单
1. 选中文本后右键
2. 点击"翻译选中文本"
3. 查看翻译结果

#### 方法三：快捷键
1. 选中文本
2. 按 Ctrl/Cmd + Shift + T
3. 查看翻译结果

### ⚙️ 技术栈

- JavaScript (ES6+)
- WebExtension API
- webextension-polyfill
- LibreTranslate API

### 📝 注意事项

1. **翻译 API**
   - 默认使用 LibreTranslate 免费实例
   - 可能有限流或不可用的情况
   - 可在设置中配置自定义 API

2. **浏览器兼容性**
   - Chrome 88+ (Manifest V3)
   - Firefox 78+ (Manifest V2)

3. **性能**
   - 相同文本会自动缓存
   - 建议分段翻译长文本
